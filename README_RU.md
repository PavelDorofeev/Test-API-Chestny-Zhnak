Здравствуйте! Это программа для тестирования АПИ маркировки честного знака в разрешительном режиме.

То есть ваша товароучетка проверяет наличие товара в обороте по запросу в честный знак.

Для этого ей нужен токен кассового аппарата, который находится у вас в личном кабинете (ЧЗ).

Чтобы запросить наименование товара вы обращаетесь в национальный каталог (ЧЗ). 
Для этого вам понадобится API KEY, который находится также в личном кабинете ЧЗ (закладка нациоанльный каталог).

После проверки товара в ЧЗ, вам еще надо будет проверить товар через канал касса - ФН- ОФД , но это уже другой этап.

Наш проект собран на Qt 4 msvc-2010 (это C++98). Подготовитель сборки qmake (если кто еще помнит такой).
Помните: никаких русских букв в пути к проекту, куда вы разместите проект (сам опять попался).

В каталоге release выложена эта же, но статически собранная без зависимостей программа, она запустится без проблем на любой ОС Windows, которая поддерживает Tls 1.2.
